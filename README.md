## SHOM•AIR — интернет-магазин кроссовок, разработанный на Django.  
### Проект включает корзину, систему пользователей и модульную архитектуру, удобную для масштабирования и дальнейшего развития.

## Особенности проекта

- Django-проект с разделением на приложения
- Приложение `main` — основная логика сайта
- Приложение `cart` — корзина и работа с товарами
- Приложение `users` — регистрация, авторизация, профиль
- Корзина покупок с контекстным процессором
- Система пользователей
- Шаблоны с наследованием
- Частичные шаблоны (partials)
- Поддержка static и media файлов
- Настройки через `.env`
- Работа проекта связана с `PostgreSQL` (База данных)

---

### Структура проекта

- `shop/`
  - `settings.py` — основные настройки проекта
  - `urls.py` — глобальные маршруты
  - `wsgi.py`, `asgi.py` — серверные интерфейсы
- `main/`
  - основное приложение сайта
- `cart/`
  - `cart.py` — логика корзины
  - `context_processors.py` — передача корзины в шаблоны
  - `views.py`, `urls.py`, `forms.py`
- `users/`
  - шаблоны пользователей:
    - `login.html`
    - `register.html`
    - `profile.html`
  - `views.py`, `forms.py`, `models.py`
- `media/` — загружаемые файлы
- `.env` — переменные окружения
- `.env.example` — пример конфигурации
- `requirements.txt` — зависимости проекта
- `manage.py` — точка входа Django

Структура соответствует best practices Django и удобна для поддержки и масштабирования.

# Запуск проекта:

### Создание и активация виртуального окружения
```
python -m venv venv
source venv/bin/activate   # Linux / Mac
venv\Scripts\activate      # Windows
```
### Установка зависимостей
```
python -m pip install -r requirements.txt
```
### Настройка переменных окружения Создайте файл .env (пример находится в .env.example):
```
SECRET_KEY=
DEBUG=False
```
### Применение миграций
```
python manage.py migrate
```
### Создание суперпользователя
```
python manage.py createsuperuser
```
# А после запуск
```
python manage.py runserver
```
